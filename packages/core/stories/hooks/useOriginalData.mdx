# `useRawData`

The `useRawData` hook returns an object holding the dataset provided to a chart view.

<Source
    language="javascript"
    code={`
import { useRawData } from '@chsk/core'

// in a JSX component:
const { data } = useRawData()
`} />

Object `data` will hold the dataset provided to the current view.
The dataset will be in the original (i.e. raw, unprocessed) format.
