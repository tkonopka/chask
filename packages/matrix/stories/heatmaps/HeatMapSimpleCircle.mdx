# HeatMapSimpleCircle

<Meta of={HeatMapSimpleCircleStories} />

`HeatMapSimpleCircle` draws a single circle.

<ArgTypes of={HeatMapSimpleCircle} />

## Example

`HeatMapSimpleCircle` is intended to be used in `HeatMapCells` to draw a heatmap with custom cells.

<Canvas of={HeatMapSimpleCircleStories.Example} />

Like `HeatMapSimpleRectangle`, `HeatMapSimpleCircle` does not provide any animations.
