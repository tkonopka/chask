# HeatMapColorRectangle

<Meta of={HeatMapColorRectangleStories} />

`HeatMapColorRectangle` draws a single rectangle.

<ArgTypes of={HeatMapColorRectangle} />

## Example

`HeatMapColorRectangle` is a drop-in replacement for `HeatMapRectangle`.

<Canvas of={HeatMapColorRectangleStories.Example} />

In contrast to `HeatMapRectangle`, `HeatMapColorRectangle` provides animation for the color fill.
Color animations can make heatmap updates appear smoother.
However, in large visualizations with hundreds of cells, animations may decrease responsiveness.
