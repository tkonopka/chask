# ViewController

<Meta title="Core/Components/Views/ViewController" />
export const Template = args => <ViewController {...args} />

`ViewController` is an organizational component that manages view state. It provides some invisible elements that can detect user interactions and a toolbar to toggle between interaction modes.

<ArgsTable of={ViewController} />

## Layout

A `ViewController` creates a toolbar with buttons.
Prop `horizontal` toggles its orientation.
