# HorizontalGoldenRectangle

`HorizontalGoldenRectangle` displays a rectangular symbol with width / height proportion set to the golden ratio (1.618).

Geometrically, golden rectangles are just rectangles and can be created using the versatile `Rectangle` component.
However, `HorizontalGoldenRectangle` is relevant in that it has the same API as symbols such as `Circle` and `Square`.
The component can be used, for example, in legends and tooltips.
