# ArcArrow

export const Template = args => <ArcArrow {...args} />

`ArcArrow` draws a curved arrow.

<ArgsTable of={ArcArrow} />

## Arrow shape

`ArcArrow` shares many props with `BlockArrow`.
In particular, props `stemWidth`, `headWidth`, and `headLength` determine the arrow shape.

<Canvas>
    <Story
        name="narrow"
        args={{
            start: [50, 160],
            end: [200, 60],
            units: 'absolute',
            stemWidth: 10,
            headWidth: 30,
            headLength: 40,
            style: { fillOpacity: 1 },
        }}
        component={ArcArrow}
        decorators={[ChartDecorator]}
    >
        {Template.bind({})}
    </Story>
    <Story
        name="wide"
        args={{
            start: [50, 160],
            end: [200, 60],
            units: 'absolute',
            stemWidth: 40,
            headWidth: 60,
            headLength: 20,
            style: { fillOpacity: 1 },
        }}
        component={ArcArrow}
        decorators={[ChartDecorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>

## Curvature and variants

Prop `r` sets a radius of curvature for the arrow body.
Prop `variant` determines the 'handed-ness' of the curvature.

<Canvas>

            end: [200, 60],

## Arrow heads

By default, `ArcArrow` creates an arrowhead only at the end position.
Arrowheads can be configured at each side via prop `heads`.

<Canvas>
    <Story
        name="double sided"
        args={{
            start: [50, 160],
            end: [200, 60],
            units: 'absolute',
            heads: [true, true],
            variant: 'left',
            r: 400,
            style: { fillOpacity: 1 },
        }}
        component={ArcArrow}
        decorators={[ChartDecorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>
