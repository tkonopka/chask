# SliceLabels

<Meta title="Addons/Polar/Pie/SliceLabels" />

export const Template = args => <SliceLabels {...args} />

`SlicesLabels` draws pieces for a pie / doughnut chart.

<ArgsTable of={SliceLabels} />

## Alignment

Prop `align` tunes the position of the label.
The default value is [0.5, 0.5], which indicates a position at the center of the slice:
halfway between the inner and outer slice radii and halfway between the start and end angles.
A value of [0.8, 0.5] pushes the label closer toward the outer edge of the slice.

## Data subsets

By default, `SlicesLabels` draws labels for all slices in a dataset. To draw subsets, set props `ids`.

<Canvas>
    <Story
        name="selected ids"
        args={{ ids: ['alpha', 'gamma'] }}
        component={SliceLabels}
        decorators={[ChartDoughnutSlicesDecorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>

## Minimum angle

Prop `minAngle` specifies an angle threshold. Labels for slices smaller than the threshold will be omitted.

<Canvas>
    <Story
        name="minimum angle"
        args={{ minAngle: 45 }}
        component={SliceLabels}
        decorators={[ChartDoughnutSlicesDecorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>
