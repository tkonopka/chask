# `useScales`

The `useScales` hook returns object related to mapping view-specific values into coordinates for drawing.

<Source
    language="javascript"
    code={`
import { useScales } from '@chsk/core'

const { scales, scaleProps, setScaleProps } = useScales()
`} />

Object `scales` contains four functions with scales for the x-axis, y-axis, color scale, and size scale.
