# HeatMapCircle

<Meta of={HeatMapCircleStories} />

`HeatMapCircle` draws a single circle. This component can be used together with `HeatMapCells` to draw a heatmap with custom cells.

<ArgTypes of={HeatMapCircle} />

`HeatMapCircle` has the same API as `HeatMapRectangle`, but uses its props to display a circle instead of a rectangle.
