# Getting started

Chsk is available on npm and on github.

## Installation

All projects using the library should install the `@chsk/core` package.

```bash
npm install @chsk/core
```

Add-on packages can be installed as needed.

```bash
npm install @chsk/annotation @chsk/band @chsk/matrix @chsk/polar @chsk/set @chsk/themes @chsk/xy
```

For information about installing a development version of the library, please see the
[contributing guide](https://github.com/tkonopka/chsk/blob/main/CONTRIBUTING.md).
