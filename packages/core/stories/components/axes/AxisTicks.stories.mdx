# AxisTicks

<Meta title="Core/Components/Axes/AxisTicks" />
export const Template = args => <AxisTicks {...args} />

`AxisTicks` draws a set of tick line and labels.
It is intended to be used a child of an `Axis` component.

<ArgsTable of={AxisTicks} />

## Variants

Prop `variant` configures ticks for the top, right, bottom, or left axis.
For brevity, the example below shows ticks on the bottom axis.

<Canvas>
    <Story
        name="default"
        args={{
            variant: 'bottom',
        }}
        component={AxisTicks}
        decorators={[ChartBottomAxisDecorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>

## Ticks

Prop `ticks` determines the position of tick marks.
The prop can be specified as an integer (to suggest a number of tick marks)
or as an array (to specify all the tick positions).

<Canvas>
    <Story
        name="number"
        args={{
            variant: 'bottom',
            ticks: 10,
        }}
        component={AxisTicks}
        decorators={[ChartBottomAxisDecorator]}
    >
        {Template.bind({})}
    </Story>
    <Story
        name="positions"
        args={{
            variant: 'bottom',
            ticks: [0, 20, 50, 90, 100],
        }}
        component={AxisTicks}
        decorators={[ChartBottomAxisDecorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>

Note that suggestions for the number of tick marks are approximate and the actual number of ticks may differ; the actual positions are computed by [d3-scale](https://github.com/d3/d3-scale).

## Labels

The position of the text labels near tick marks is controlled with props `labelDistance` and `labelOffset`.

`labelDistance` sets a perpendicular distance from the axis to the text label.
`labelOffset` further translates the label in an arbitrary direction.

<Canvas>
    <Story
        name="label distance"
        args={{
            variant: 'bottom',
            labelDistance: 25,
        }}
        component={AxisTicks}
        decorators={[ChartBottomAxisDecorator]}
    >
        {Template.bind({})}
    </Story>
    <Story
        name="label offset"
        args={{
            variant: 'bottom',
            labelDistance: 5,
            labelOffset: [10, 5],
        }}
        component={AxisTicks}
        decorators={[ChartBottomAxisDecorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>

## Rotation

Labels can be rotated using prop `labelAngle`.
Note that adjusting the angle will likely prompt adjustment to css properties like `textAnchor` and `dominantBaseline`.

<Canvas>
    <Story
        name="rotation"
        args={{
            variant: 'bottom',
            labelDistance: 10,
            labelAngle: -90,
            labelStyle: {
                textAnchor: 'end',
                dominantBaseline: 'middle',
            },
        }}
        component={AxisTicks}
        decorators={[ChartBottomAxisDecorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>

## Label format

Prop `labelFormat` specifies a function that converts a tick value into a string label.
This can be used to create custom label, for example, to include percentage marks.

<Canvas>
    <Story
        name="format"
        args={{
            variant: 'bottom',
            labelDistance: 10,
            labelFormat: v => v + '%',
        }}
        component={AxisTicks}
        decorators={[ChartBottomAxisDecorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>
