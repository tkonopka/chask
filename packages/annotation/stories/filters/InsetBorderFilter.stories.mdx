# InsetBorderFilter

export const Template = args => <InsetBorderFilter {...args} />

`InsetBorderFilter` creates an svg filter that applies a solid color mask to the boundary of a target.

<ArgsTable of={InsetBorderFilter} />

## Color and opacity

Prop `floodColor` determines the color of the mask.
Prop `floodOpacity` determines the intensity of the filter.

## Inset

By default, `InsetBorderFilter` affects the entire shape, including any existing boundary.
To exclude the boundary from the filter, set prop `erodeR` with the width of the boundary stroke.
