# Square

<Meta of={SquareStories} />

`Square` displays a square shape.

Geometrically, squares are a special types of rectangles.
They can be created using the versatile `Rectangle` component.
However, `Square` is relevant in that its API is equivalent to `Circle` and other symbols like `Triangle` or `Diamond`.
The component can be used, for example, in legends.
