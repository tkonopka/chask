# BlockArrow

export const Template = args => <BlockArrow {...args} />

`BlockArrow` displays a thick arrow.

<ArgsTable of={BlockArrow} />

## Arrow shape

The shape of the arrow is determined by props `stemWidth`, `headWidth`, and `headLength`.

<Canvas>
    <Story
        name="narrow"
        args={{
            start: [50, 160],
            end: [200, 60],
            units: 'absolute',
            stemWidth: 10,
            headWidth: 30,
            headLength: 40,
            style: { fillOpacity: 1 },
        }}
        component={BlockArrow}
        decorators={[ChartDecorator]}
    >
        {Template.bind({})}
    </Story>
    <Story
        name="wide"
        args={{
            start: [50, 160],
            end: [200, 60],
            units: 'absolute',
            stemWidth: 40,
            headWidth: 60,
            headLength: 20,
            style: { fillOpacity: 1 },
        }}
        component={BlockArrow}
        decorators={[ChartDecorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>

## Arrow heads

By default, `BlockArrow` creates a shape that is blunt at the start position and has an arrowhead at the end position.
Arrowheads can be configured at each side via prop `heads`.

<Canvas>
    <Story
        name="double sided"
        args={{
            start: [50, 160],
            end: [200, 60],
            units: 'absolute',
            heads: [true, true],
            style: { fillOpacity: 1 },
        }}
        component={BlockArrow}
        decorators={[ChartDecorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>

## Styling

Arrows can be styled using css, including with gradients and filters.

<Canvas>
    <Story
        name="css style"
        args={{
            start: [50, 160],
            end: [200, 60],
            units: 'absolute',
            stemWidth: 12,
            style: { strokeWidth: 2, stroke: '#dd0000', fill: '#dd7777', fillOpacity: 1 },
        }}
        component={BlockArrow}
        decorators={[ChartDecorator]}
    >
        {Template.bind({})}
    </Story>
    <Story name="gradient">
        <Chart size={[400, 300]} padding={[40, 40, 40, 40]} style={{ display: 'inline-block' }}>
            <defs>
                <LinearGradient
                    id="arrow-gradient"
                    start={[0, 1]}
                    end={[1, 0.333]}
                    stops={['#dd000000', '#dd0000ff']}
                    offsets={[0.05, 0.4]}
                />
            </defs>
            <Surface variant={'inner'} />
            <BlockArrow
                stemWidth={12}
                start={[50, 160]}
                end={[200, 60]}
                units={'absolute'}
                style={{ fill: 'url(#arrow-gradient)', strokeWidth: 0, fillOpacity: 1 }}
            />
        </Chart>
    </Story>
    <Story name="filter">
        <Chart size={[400, 300]} padding={[40, 40, 40, 40]} style={{ display: 'inline-block' }}>

                end={[200, 60]}
                units={'absolute'}

                    fill: '#dd0000',
                    fillOpacity: 1,
                    filter: 'url(#arrow-shadow)',
                    strokeWidth: 0,
                }}
            />
        </Chart>
    </Story>
</Canvas>
