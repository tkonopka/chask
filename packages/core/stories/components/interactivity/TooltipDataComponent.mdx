# TooltipDataComponent

<Meta title="Core/Components/Interactivity/TooltipDataComponent" />

`TooltipDataComponent` is a wrapper function that associates event handlers with components.

`TooltipDataComponent` is a drop-in replacement for `DataComponent`, and it differs from `DataComponent`
in that it manages state of the tooltip context.
