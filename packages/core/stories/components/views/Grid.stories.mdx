# Grid

<Meta title="Core/Components/Views/Grid" />
export const Template = args => <Grid {...args} />

`Grid` is an organizational component.
It does not display any visual elements, but it conceptually splits the available chart space into a grid.
Children components can access this grid to position their content.

<ArgsTable of={Grid} />

## Rows and columns

Prop `grid` accepts an array with integers representing the number of columns and rows in the grid.

The example below creates a 3x3 grid, and populates the cells with content to visualize the layout.

<Canvas>
    <Story
        name="grid size"
        args={{
            grid: [3, 3],
            children: <GridFiller3x3 />,
        }}
        component={Grid}
        decorators={[ChartPlainDecorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>

## Variants

Prop `variant` toggles between horizontal and vertical arrangements of cells within the grid.
The default layout is `horizontal`, in which the cells are arranged from left-to right, then down a row, and so on
(see examples above).
The alternative variant is `vertical`, in which the cells are arranged going downward, then across.

<Canvas>
    <Story
        name="vertical"
        args={{

            children: <GridFiller3x3 />,
        }}
        component={Grid}
        decorators={[ChartPlainDecorator]}

        {Template.bind({})}

</Canvas>
