# Distributions

<Meta of={DistributionsStories} />

`Distributions` summarizes distributions of numeric values.

`Distributions` is intended to be used within a `Distribution` view.
Note that while the `Distributions` component is the one that draws boxes and whiskers, much of the chart layout is determined by the parent view `Distribution`.

<ArgTypes of={DistributionsStories} />

## Component

Prop `component` specifies how distributions are portrayed graphically on the chart.
By default, `Distributions` uses a `BoxAndWhiskers` component.

<Canvas of={DistributionsStories.BoxesVertical} />
<Canvas of={DistributionsStories.BoxesHorizontal} />

Alternative components are `BarAndWhisker` and `LineAndWhiskers`.

<Canvas of={DistributionsStories.BarsVertical} />
<Canvas of={DistributionsStories.BarsHorizontal} />
<Canvas of={DistributionsStories.LinesVertical} />
<Canvas of={DistributionsStories.LinesHorizontal} />

Note that `BoxAndWhiskers` always uses non-parametric summary statistics (quantiles).
In contrast, `BarAndWhisker` and `LineAndWhiskers` use parametric summary statistics (mean and standard deviation).

Prop `component` also accepts custom implementations of the `BoxAndWhiskersProps` interface.

## Data subsets

By default, `Distributions` draws the entire dataset.
To draw subsets, use props `ids` and `keys`.

<Canvas of={DistributionsStories.SelectedIds} />
<Canvas of={DistributionsStories.SelectedKeys} />

## Whisker cap width

The size of the whisker 'cap' can be adjusted with prop `whiskerCapWidth`.

<Canvas of={DistributionsStories.WhiskerCap} />
