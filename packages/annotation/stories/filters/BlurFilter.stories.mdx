# BlurFilter

export const Template = args => <BlurFilter {...args} />

`BlurFilter` creates an svg filter that applies a blur effect.

<ArgsTable of={BlurFilter} />

(`BlurFilter` is a simple a wrapper for svg's `feGaussianBlur`.
The filter is provided as a distinct component for consistency in notation with other library filters.)

## Blur

Prop `blurStdDeviation` determines the amount of blur.
