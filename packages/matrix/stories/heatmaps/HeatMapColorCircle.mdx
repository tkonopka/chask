# HeatMapColorCircle

<Meta of={HeatMapColorCircleStories} />

## Example

`HeatMapColorCircle` is closely related to `HeatMapCircle`.

<Canvas of={HeatMapColorCircleStories.Example} />

In contrast to `HeatMapCircle`, `HeatMapColorCircle` provides animation for the color fill.
Color animations can make heatmap updates appear smoother.
However, in large visualizations with hundreds of cells, animations may decrease responsiveness.
