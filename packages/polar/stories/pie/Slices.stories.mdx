# Slices

<Meta title="Addons/Polar/Pie/Slices" />

export const Template = args => <Slices {...args} />

`Slices` draws pieces for a pie / doughnut chart.

<ArgsTable of={Slices} />

## Data subsets

By default, `Slices` draws an entire dataset. To draw subsets, set props `ids`.

<Canvas>
    <Story
        name="selected ids"
        args={{ ids: ['alpha', 'gamma'] }}
        component={Slices}
        decorators={[ChartPieDecorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>

## Radii

Props `rInner` and `rOuter` determine the radii of slices.
Default values are `rInner=0` and `rOuter=1`, which create slices for a pie chart.
To create a doughnut, set `rInner` to a non-zero value.

<Canvas>
    <Story
        name="pie"
        args={{
            rInner: 0,
            rOuter: 1,
        }}
        component={Slices}
        decorators={[ChartPieDecorator]}
    >
        {Template.bind({})}
    </Story>
    <Story
        name="doughnut"
        args={{
            rInner: 0.6,
            rOuter: 1,
        }}
        component={Slices}
        decorators={[ChartPieDecorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>

Prop `rOuter` determines the overall scale.

<Canvas>
    <Story
        name="scaled"
        args={{
            rInner: 0.4,
            rOuter: 0.6,
        }}
        component={Slices}
        decorators={[ChartPieDecorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>

## Decorations

Prop `rCorner` creates slices with rounded corners.
Prop `padAngle` creates empty space between adjacent slices.

<Canvas>
    <Story
        name="rounded corners"
        args={{
            rCorner: 5,
            rInner: 0.8,
            rOuter: 1,
            padAngle: 0.05,
        }}
        component={Slices}
        decorators={[ChartPieDecorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>
