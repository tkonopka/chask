# VennSetLabels

`VennSetLabels` draws labels on a Venn chart.

## Data subsets

By default, `VennSetLabels` draws labels for all the sets in the dataset.
To draw subsets, specify the desired sets via prop `ids`.

<Canvas of={VennSetLabelsStories.Default} />
<Canvas of={VennSetLabelsStories.OneSet} />

## Label position

By default, labels appear at the centers of the sets.
The positions can be adjusted with props `rs`, `angles`, and `offset`.
The first two props displace the labels in a radial and angular direction.
The radial distance is relative to the size of the sets, so labels with unit radii are placed on / near the circumference.
Translations are absolute shifts.

<Canvas of={VennSetLabelsStories.LabelsUnderSets} />

For labels that require different translations, use separate `VennSetLabels` components, each with its own custom settings.

## Label format

By default, labels display the set id. The content can be adjusted via prop `format`.
