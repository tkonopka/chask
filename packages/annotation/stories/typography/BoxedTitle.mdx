# BoxedTitle

`BoxedTitle` displays a text label positioned around the edges of a chart.

<ArgTypes of={BoxedTitleStories} />

## Variants

Prop `variant` determines whether the labels is placed on the top, right, bottom, or left side of a chart.

<Canvas of={BoxedTitleStories.Top} />
<Canvas of={BoxedTitleStories.Right} />
<Canvas of={BoxedTitleStories.Left} />
<Canvas of={BoxedTitleStories.Bottom} />

## Distance

Boxes can be displaced away from the chart boundary using prop `distance`.

<Canvas of={BoxedTitleStories.Distance} />

## Expansion

Prop `expansion` takes adjusts the extent of the box.

<Canvas of={BoxedTitleStories.SmallExpansion} />
<Canvas of={BoxedTitleStories.LargeExpansion} />

Note a small negative expansion will shrink the box, and can balance a non-zero color border.

## Corners

Box corners can be rounded using props `rx` and `ry`.

<Canvas of={BoxedTitleStories.RoundedCorners} />

## Content

The label can contain arbitrary content.

<Canvas of={BoxedTitleStories.CustomContent} />

## Styling

It is possible to style the box as well as the text.

<Canvas of={BoxedTitleStories.Styling} />
