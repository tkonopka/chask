# Typography

<Meta title="Core/Components/Text/Typography" />
export const Template = args => <Typography {...args} />

`Typography` places a text element on a chart.

<ArgsTable of={Typography} />

## Variants

There are a number of predefined variants, which can be activated via the prop `variant`.

<Canvas>
    <Story
        name="title"
        args={{
            variant: 'title',
            position: [0, -15],
            children: 'Primary heading',
        }}
        component={Typography}
        decorators={[ChartDecorator]}
    >
        {Template.bind({})}
    </Story>
    <Story
        name="subtitle"
        args={{
            variant: 'subtitle',
            position: [0, -15],
            children: 'Secondary heading',
        }}
        component={Typography}
        decorators={[ChartDecorator]}
    >
        {Template.bind({})}
    </Story>
    <Story
        name="axisLabel"
        args={{
            variant: 'axisLabel',
            position: [160, 235],
            children: 'axis label',
        }}
        component={Typography}
        decorators={[ChartDecorator]}
    >
        {Template.bind({})}
    </Story>
    <Story
        name="tickLabel"
        args={{
            variant: 'tickLabel',
            position: [0, 235],
            children: 'tick label',
        }}
        component={Typography}
        decorators={[ChartDecorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>

In the examples above, the prop `variant` determines the visual style for the text.
Positions are set manually to suggest how the visual styles might be used within a chart.

## Rotation

Text can be rotated with prop `angle`.

<Canvas>
    <Story
        name="rotation"
        args={{
            variant: 'default',
            position: [120, 40],
            children: 'rotated text',
            style: {
                textAnchor: 'middle',
                dominantBaseline: 'middle',
            },
            angle: -10,
        }}
        component={Typography}
        decorators={[ChartDecorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>

## Styles

Components can be adjusted individually using css styles.

<Canvas>
    <Story
        name="color"
        args={{
            variant: 'title',
            position: [0, -20],
            children: 'Title in color',
            style: {
                fill: '#dd4444',
                dominantBaseline: 'middle',
            },
        }}
        component={Typography}
        decorators={[ChartDecorator]}
    >
        {Template.bind({})}
    </Story>
    <Story
        name="letter spacing"
        args={{
            variant: 'default',
            position: [0, -20],
            children: 'Text with custom style',
            style: {
                textAnchor: 'start',
                fill: '#222222',
                fontFamily: 'serif',
                letterSpacing: 3,
                fontSize: '20px',
                dominantBaseline: 'middle',
            },
        }}
        component={Typography}
        decorators={[ChartDecorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>

## Content

The component accepts complex children, including styled `<tspan>` elements.

<Canvas>
    <Story
        name="tspan"
        args={{
            variant: 'default',
            position: [40, 40],
            style: { textAnchor: 'start' },
            children: (
                <tspan>
                    This is <tspan style={{ fontWeight: 600 }}>bold</tspan>
                </tspan>
            ),
        }}
        component={Typography}
        decorators={[ChartDecorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>
