# Line

<Meta of={LineStories} />

`Line` draws a single line on a chart.

<ArgTypes of={LineStories} />

## Variants

Prop `variant` applies pre-defined styles to a line.
The examples below show variants `tick` and `grid`, which apply styling suitable for axis tick marks
and for grid lines, respectively.
