# LineLabel

export const Template = args => <LineLabel {...args} />

`LineLabel` displays a line segment and a text label.

<ArgsTable of={LineLabel} />

## Band labels

`LineLabel` can be used on band scales to designate groups. Props `start` and `end` control the extent of the line segment.
By default, line edges coincide with the centers of the designated bands, but they can be extended via a prop `extension`.

<Canvas>
    <Story
        name="band centers"
        args={{
            start: ['A', 104],
            end: ['D', 104],
            angle: 0,
            textStyle: { textAnchor: 'middle' },
            children: 'A - D',
        }}
        component={LineLabel}
        decorators={[ChartBandViewAxisDecorator]}
    >
        {Template.bind({})}
    </Story>
    <Story
        name="band edges"
        args={{
            start: ['A', 104],
            end: ['D', 104],

            angle: 0,
            textStyle: { textAnchor: 'middle' },
            children: 'A - D (extended)',
        }}
        component={LineLabel}
        decorators={[ChartBandViewAxisDecorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>

## Text positioning

By default, the text label is placed at the center of the interval. The label can be aligned along the line using prop `align` and displaced from the line using prop `offset`.

<Canvas>
    <Story
        name="left aligned"
        args={{
            start: ['B', 104],
            end: ['E', 104],
            align: 0,
            expansion: [0.5, 0.5],
            textStyle: { textAnchor: 'start' },
            children: 'left aligned',
        }}
        component={LineLabel}
        decorators={[ChartBandViewAxisDecorator]}
    >
        {Template.bind({})}
    </Story>
    <Story
        name="below line"
        args={{

            end: ['F', -16],

            expansion: [0.5, 0.5],
            textStyle: { textAnchor: 'middle' },
            children: 'below line',
        }}
        component={LineLabel}
        decorators={[ChartBandViewAxisDecorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>

## Styling

Both the line and the text can be styled using css. The line accepts arrow markers.
