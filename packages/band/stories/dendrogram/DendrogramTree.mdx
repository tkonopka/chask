# DendrogramTree

`DendrogramTree` draws a dendrogram hierarchy.

<ArgTypes of={DendrogramTreeStories} />

## Subtrees

By default, `DendrogramTree` draws the entire dendrogram hierarchy.

<Canvas of={DendrogramTreeStories.AllNodes} />

To draw a subset, specify the specific branches using prop `levels`.
Alternatively, to draw a subtree, specify prop `keys` with selected leaf node identifiers.

<Canvas of={DendrogramTreeStories.Levels} />
<Canvas of={DendrogramTreeStories.Keys} />
