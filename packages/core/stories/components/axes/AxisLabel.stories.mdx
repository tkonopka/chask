# AxisLabel

<Meta title="Core/Components/Axes/AxisLabel" />
export const Template = args => <AxisLabel {...args} />

`AxisLabel` draws text intended as a label for an axis.

<ArgsTable of={AxisLabel} />

`AxisLabel` should be used within an `Axis` component.

## Variants

Predefined variants place a label near the top, right, bottom, or left axis. For brevity, the examples below shows labels on the bottom and left axes.

<Canvas>
    <Story
        name="bottom axis"

        }}
        component={AxisLabel}

    >
        {Template.bind({})}
    </Story>
    <Story
        name="left axis"

</Canvas>

## Distance from baseline

The distance between the axis line and the axis label is controlled by prop `distance`.
The value can be positive or negative, moving the label away or toward the center, respectively.

<Canvas>
    <Story
        name="negative distance"
        args={{
            variant: 'bottom',
            children: 'axis label',
            distance: -10,
        }}
        component={AxisLabel}
        decorators={[ChartBottomAxisDecorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>

## Alignment

The position of the label along the axis is controlled by prop `align`, which
here takes a single value from 0 (start) to 1 (end).

<Canvas>
    <Story
        name="alignment"
        args={{
            variant: 'bottom',
            children: 'axis label',
            align: 1,
            style: {
                textAnchor: 'end',
            },
        }}
        component={AxisLabel}
        decorators={[ChartBottomAxisDecorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>

Note that the value of this prop is often adjusted together with the css style `textAnchor`,
as in the example above. Another prop to fine-tune the label position is `offset`, which
translates the label in an arbitrary direction.
