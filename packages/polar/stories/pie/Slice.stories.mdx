# Slice

<Meta title="Addons/Polar/Pie/Slice" />

export const Template = args => <Slice {...args} />

`Slice` draws one piece of a pie / doughnut chart.

`Slice` components are typically not used as standalone objects, but rather used as a prop to `Slices`.

## Padding

Prop `padAngle` shrinks the effective shape to create empty space at the slice boundaries.
The examples below display slices that form semi-circles.
Non-zero `padAngle` values make the slices appear slightly smaller.
